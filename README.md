# Dotfiles

Shell configuration for dev containers and local dev on macos/linux.

- Shell: [fish](https://fishshell.com)
- Editor: [helix](https://helix-editor.com)
- VCS: [git](https://git-scm.com/docs) + [jj](https://martinvonz.github.io/jj/latest)
- Package manager: [brew](https://brew.sh)
- Rust version manager: [rustup](https://rust-lang.github.io/rustup/concepts/index.html)
- Dotnet version manager: brew tap [isen-ng/dotnet-sdk-versions](https://github.com/isen-ng/homebrew-dotnet-sdk-versions)
- Node.js version manager: [volta](https://docs.volta.sh/guide)
- Presentations in terminal: [slides](https://github.com/maaslalani/slides)
